# UBI実現可能性シミュレーター

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://github.com/kainosuke1999/ubi-calculator/blob/main/LICENSE) [![GitHub last commit](https://img.shields.io/github/last-commit/kainosuke1999/ubi-calculator.svg)](https://github.com/kainosuke1999/ubi-calculator/commits/main)

**AI時代におけるユニバーサル・ベーシック・インカム（UBI）の実現可能性を定量的に評価するシミュレーター**

## 📜 プロジェクトの背景

急速なAI技術の進展は、労働市場に構造的な変化をもたらし、多くの仕事が自動化される可能性が指摘されています。このような社会変動の中で、全ての人々の生活を保障するセーフティネットとして、ユニバーサル・ベーシック・インカム（UBI）への関心が高まっています。

しかし、UBIの財源確保は常に大きな課題でした。本プロジェクトは、AIの生産性向上効果を財源の一部と捉え、日本におけるUBIの財政的実現可能性を客観的かつ定量的に評価するために立ち上げられました。

## 🌟 プロジェクトの目的

本プロジェクトは、以下の目的を達成するための動的経済モデルとシミュレーターを提供します。

- **客観的なデータに基づく議論**: UBIに関する感情的・観念的な議論から脱却し、データに基づいた建設的な対話の土台を築く。
- **政策決定の支援**: 政策立案者や研究者が、様々な経済シナリオの下でUBIの持続可能性を評価できるツールを提供する。
- **社会への問題提起**: 「AIと共存する未来の社会保障」という重要なテーマについて、広く社会の関心を喚起する。

## 📋 概要

本シミュレーターは、従来の静的モデルとは異なり、UBI導入が経済に与える動的な影響を考慮した「動的社会維持コストモデル」を採用しています。

- **動的社会維持コスト**: UBI給付額に応じて、既存の社会保障費（生活保護、年金の一部など）が動的に削減される効果をモデル化。
- **税率感応度**: 法人税率の変動が企業の生産性や投資意欲に与える負の影響を考慮。
- **インフレ率の考慮**: UBI給付総額が名目GDPに占める割合に応じて、インフレ率が変動するメカニズムを導入。
- **実質購買力の算出**: インフレの影響を調整し、名目の給付額だけでなく、実質的な購買力を評価。

## 📊 主要な発見

論文「AI時代におけるユニバーサル・ベーシック・インカムの実現可能性：動的社会維持コストモデルによる定量的分析」で示された主要な結果は以下の通りです。

| シナリオ | AI代替率 | 法人税率 | 月額UBI（名目） | 月額UBI（実質） | インフレ率 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 現状（2025年） | 5% | 30% | 給付不可能 | - | - |
| 中程度AI進展 | 30% | 50% | 13.1万円 | 12.5万円 | 5.0% |
| 高度AI進展 | 60% | 70% | 41.1万円 | 37.1万円 | 10.8% |
| 提案ケース | 80% | 80% | 70.7万円 | 61.8万円 | 14.4% |

**重要な知見**: 法人税率を単純に引き上げればUBI財源が増えるわけではなく、税率が生産性に与える負の影響を考慮した最適税率の探索が重要であることを示唆しています。

## 🛠️ 技術スタック

### モバイルアプリ

- **フレームワーク**: React Native + Expo SDK 54
- **言語**: TypeScript 5.9
- **UI**: NativeWind 4 (Tailwind CSS)
- **状態管理**: React Context + useReducer
- **テスト**: Vitest

### 計算エンジン

- **言語**: TypeScript
- **数値計算**: 5回の反復計算による収束アルゴリズム
- **パラメーター**: 10以上の調整可能な経済変数

## 💻 ローカル開発

### 前提条件

- Node.js 22.x
- pnpm 9.x
- Expo CLI

### セットアップ

```shell
# リポジトリをクローン
git clone https://github.com/kainosuke1999/ubi-calculator.git
cd ubi-calculator/app

# 依存関係をインストール
pnpm install

# 開発サーバーを起動
pnpm dev

# テストを実行
pnpm test
```

### アプリをモバイルで確認

1.  スマートフォンに **Expo Go** アプリをインストールします。
2.  開発サーバー起動後に表示されるQRコードをスキャンすると、アプリが起動します。
