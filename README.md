# UBI実現可能性シミュレーター

**AI時代におけるユニバーサル・ベーシック・インカム（UBI）の実現可能性を定量的に評価するシミュレーター**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

---

## 📋 概要

本プロジェクトは、人工知能（AI）技術の発展が労働市場に与える影響を考慮し、日本におけるUBIの財政的実現可能性を評価するための動的経済モデルとシミュレーターを提供します。

従来の静的モデルとは異なり、本モデルは以下の特徴を持ちます：

- **動的社会維持コスト**：UBI給付額に応じて社会保障費が動的に削減される
- **税率感応度**：法人税率が生産性に与える負の影響を考慮
- **インフレ率の考慮**：UBI給付総額がGDPに占める割合に応じたインフレ率を計算
- **実質購買力の算出**：名目UBI給付額からインフレ調整後の実質購買力を表示

## 📊 主要な発見

論文「AI時代におけるユニバーサル・ベーシック・インカムの実現可能性：動的社会維持コストモデルによる定量的分析」で示された主要な結果：

| シナリオ | AI代替率 | 法人税率 | 月額UBI（名目） | 月額UBI（実質） | インフレ率 |
|---------|---------|---------|----------------|----------------|-----------|
| 現状（2025年） | 5% | 30% | 給付不可能 | - | - |
| 中程度AI進展 | 30% | 50% | 13.1万円 | 12.5万円 | 5.0% |
| 高度AI進展 | 60% | 70% | 41.1万円 | 37.1万円 | 10.8% |
| 提案ケース | 80% | 80% | 70.7万円 | 61.8万円 | 14.4% |

**重要な知見**：法人税率を引き上げれば税収が増えてUBI給付可能額が増えるという単純な関係ではなく、税率が生産性に与える負の影響を考慮した最適化が必要。

## 🧮 モデルの数式

### 1. GDP計算
```
GDP = GDP_base × m × γ
```

### 2. 生産性の内生化（税率感応度）
```
γ = γ0 × (1 - λ · t_corporate)
```
- λ（税率感応度）= 0.3

### 3. インフレ率
```
π = η · (Total_UBI / GDP)
```
- η（インフレ感応度）= 0.2

### 4. 動的社会保障削減率
```
r = min(UBI_annual / 1,000,000, 1.0) × ρ
```

詳細は[論文](paper/ubi-paper-ssrn.pdf)を参照してください。

## 💻 ローカル開発

### 前提条件
- Node.js 22.x
- pnpm 9.x
- Expo CLI

### セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/[username]/ubi-calculator.git
cd ubi-calculator/app

# 依存関係をインストール
pnpm install

# 開発サーバーを起動
pnpm dev

# テストを実行
pnpm test
```

## 📱 使い方

1. **パラメーターを調整**：AI労働代替率、法人税率、社会保障削減率などを調整
2. **計算を実行**：リアルタイムでUBI給付可能額を計算
3. **結果を確認**：名目UBI、実質UBI、インフレ率、純財政余剰などを表示
4. **プリセットシナリオ**：論文の4シナリオをワンタップで再現
