# UBI実現可能性シミュレーター - デザイン計画

## アプリの目的
AI時代におけるユニバーサルベーシックインカム（UBI）の実現可能性を、経済パラメーターを調整しながらリアルタイムでシミュレーションできるモバイルアプリケーション。

## 画面構成

### 1. ホーム画面（メインシミュレーター）
**主要コンテンツ:**
- パラメーター入力エリア（スライダー）
  - AI労働代替率 (α): 0-100%
  - 資本収益集中度 (β): 0-100%
  - 生産性向上率 (γ): 1.0-2.0倍
  - 個人所得税率: 0-50%
  - 法人税率: 0-100%
  - 消費税率: 0-30%
  - 固定資産税率: 0-5%

- 結果表示エリア
  - 月額UBI給付可能額（大きく表示）
  - 年額UBI給付可能額
  - 総税収（GDP比）
  - 純財政余剰（GDP比）
  - 税収内訳（円グラフまたは棒グラフ）

**ユーザーフロー:**
1. アプリを開く → メインシミュレーター画面
2. スライダーでパラメーターを調整
3. リアルタイムで結果が更新される
4. プリセットシナリオ（現状、中程度AI進展、高度AI進展、提案ケース）を選択可能

### 2. 情報画面（オプション）
**主要コンテンツ:**
- パラメーターの説明
- モデルの前提条件
- 参考文献

## デザイン原則

### カラーパレット
- プライマリーカラー: ブルー系（経済・信頼性を表現）
- アクセントカラー: グリーン系（UBI給付可能額の表示に使用）
- 背景: ライト/ダークモード対応

### タイポグラフィ
- 見出し: 太字、大きめ
- 数値: モノスペースフォント、視認性重視
- 説明文: 読みやすいサイズ

### レイアウト
- 縦スクロール可能
- パラメーター入力は上部
- 結果表示は下部
- 片手操作を考慮したUI配置

## 技術的考慮事項
- React Native + Expo
- ローカル計算（サーバー不要）
- AsyncStorageでパラメーター設定を保存
- react-native-chartでグラフ表示（オプション）
